# 🐍 Snake Game (Python)

A classic Snake Game built using Python Turtle Graphics. This project demonstrates Object-Oriented Programming (OOP), collision detection, score tracking, and game logic using Python.

---

## ✨ Features

- Smooth snake movement
- Random food generation
- Score tracking
- Snake grows after eating food
- Wall collision detection
- Self collision detection
- Object-Oriented Programming (OOP)

---

## 🛠️ Technologies Used

- Python 3
- Turtle Graphics

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/deepeshashwinp/Snake-Game-Python.git
```

2. Open the project folder

```bash
cd Snake-Game-Python
```

3. Run the game

```bash
python main.py
```

## 🎮 Controls

- ⬆️ Up Arrow – Move Up
- ⬇️ Down Arrow – Move Down
- ⬅️ Left Arrow – Move Left
- ➡️ Right Arrow – Move Right
